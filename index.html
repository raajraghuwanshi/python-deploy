    About This Project

    Why SENTRY exists, what powers it, and where it goes next.

    Objective

    Turn raw Chicago crime records into a practical safety tool: score every district and time window by risk, and translate that score into a plain-language travel recommendation someone can actually use.

    Dataset Used

    City of Chicago public crime dataset — incident-level records including district, date and time, crime type, and location, cleaned and time-binned into five daily slots for analysis.

    Technologies

    Python Pandas NumPy Matplotlib Seaborn Scikit-learn HTML5 CSS3 JavaScript

    Future Scope

    Live API ingestion for real-time scoring, GPS-based route risk mapping, a mobile companion app, and a predictive model that forecasts risk ahead of time instead of only reporting on history.
